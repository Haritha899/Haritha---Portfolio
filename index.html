Projects

Deadlock – 3D Escape Room with Face-Swapping Integration

Deadlock is an interactive 3D escape room game developed in Unity where players solve logical puzzles within a limited time to progress through immersive levels. The system is designed as a tech-enabled escape room experience that combines entertainment with potential applications in gamified learning, training environments, and collaborative problem-solving.

A key innovation of this project is the integration of face-swapping technology, allowing players to embody in-game characters. This feature enhances realism, personalization, and engagement while introducing an advanced human–computer interaction element to traditional escape room gameplay.

My Contribution: Led the complete UI design and implementation, developing interactive panels, menus, and player interface elements. I was primarily responsible for level scripting and scene logic, ensuring smooth gameplay progression, puzzle flow, and efficient scene transitions.

Technologies: Unity, C#, UI Canvas, Scene Management

Automatic Exam Seating Arrangement System

The Automatic Exam Seating Arrangement System is a web-based application developed to automate the allocation of exam seats in an organized and efficient manner. The system reduces manual administrative workload, minimizes human errors during exam preparation, and ensures structured seating plans through optimized allocation logic.

The platform provides a user-friendly graphical interface for entering class and room details, including student lists categorized by semester and department, room capacities, and seating layouts. It supports different exam types such as semester and series exams, applying distinct algorithms to optimize seat distribution while ensuring fairness, security, and proper arrangement of students from different departments in adjacent seating.

My Contribution: Designed and developed the frontend interface using HTML and CSS, structured input handling for class and room data, implemented backend logic using Flask, and contributed to database integration for optimized seat allocation. I also documented the project comprehensively using LaTeX.

Technologies: HTML, CSS, Flask, MySQL
